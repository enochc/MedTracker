You'll need to provide at least 2 screenshots:
- Minimum: 2 screenshots
- Format: PNG or JPEG
- Dimensions: Between 320px and 3840px on shortest side
- Recommended: 1080x1920 (portrait) or 1920x1080 (landscape)

**How to create screenshots:**
1. Run the app on your Android device or emulator
2. Take screenshots of:
   - Home screen showing medication list
   - Widget on home screen
   - Medication confirmation dialog
   - History screen (optional)
3. You can use Android Studio's built-in screenshot tool
